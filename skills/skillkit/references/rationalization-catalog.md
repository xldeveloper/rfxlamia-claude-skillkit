# Rationalization Catalog

Pre-built rationalization tables for common skill types.

## Discipline Skills (e.g., TDD)

| Rationalization | Why It's Wrong | Counter in Skill |
|-----------------|----------------|------------------|
| "Too simple to test" | Simple code breaks, tests take 30s | "Simple code breaks. Test takes 30 seconds." |
| "I'll test after" | Tests-after = "what does this do?" not "what should this do?" | "Tests-after proves nothing." |
| "About the spirit" | Letter violation = spirit violation | "Violating letter = violating spirit." |
| "Just reference" | Reference = looking = cheating | "Delete means delete. Don't look." |
| "Just this once" | Exceptions destroy discipline | "No exceptions. Not once." |
| "Already manually tested" | Manual != automated, not repeatable | "Manual testing doesn't count." |
| "Different because..." | Rationalization pattern detected | "All of these mean: Start over." |
