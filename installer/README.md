# @rfxlamia/skillkit

Interactive CLI installer for SkillKit — Claude Code skills & agents.

## Usage

```bash
npx @rfxlamia/skillkit install
```

## What it does

- Lists all available skills and agents from the SkillKit repository
- Lets you pick which ones to install
- Supports both user scope (`~/.claude/skills/`) and project scope (`./.claude/skills/`)
- Copies files directly — no git cloning required
